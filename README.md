## AXIS について

AXIS は、もともと 2001 年から慶應義塾大学 ローバースカウト の海外派遣組織として活動してきたグループが、**2024 年に学生の国際協力団体として独立**した約 50 名規模の組織です。

モンゴル · マレーシア · ネパールなどを中心に、**現地の課題を自分たちで定義し、0 から設計し、実行し、継続する**活動を行っています。

### これまでの主な取り組み (2024 · 2025)

- **青年リーダーシップ教育 キャンプ建設** (モンゴル)
- **植林 300 本** — クラウドファンディング + ユニセフ支援により実施
- 環境教育プログラム · 現地 NGO 連携の職業教育
- 現地参加者 延べ **200 名以上**

### 2026 年の方針

- 「一回限りではない、持続的なパートナーシップ」を運営の軸に
- 特定の学校 · 団体との連続活動へ移行
- 国際スカウトネットワークを活用した活動圏の拡張 (香港 · フィリピンなど)
