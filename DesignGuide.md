# Axis Design System Guide: "The Global Connection"

One Tree Planted의 신뢰감과 명확성을 계승하되, **Axis**만의 글로벌 협력과 진정성을 강조할 수 있는 더 깊이 있고 프리미엄한 디자인 가이드를 제안합니다.

## 1. Design Concept: "Heroic Legacy"
공식 로고의 **인장(Seal/Emblem)** 형태에서 느껴지는 신뢰와 전통을 현대적인 UI와 결합합니다. Axis는 단순한 봉사 단체를 넘어, 명확한 정체성과 역사를 가진 '글로벌 연구/봉사 기구'로서의 이미지를 구축합니다.

---

## 2. Updated Color Palette (Logo-Unified)
로고에서 직접 추출한 색상을 웹사이트 전반의 메인/서브 컬러로 완벽히 통일합니다.

- **Main Color (Axis Navy):** `#11235A`
  - 의미: 브랜드의 신뢰도, 국제적 협력, 권위와 안정감.
  - 활용: 내비게이션 바, 히어로 배경 레이어, 주요 타이틀, 푸터.
- **Sub Color (Axis Gold):** `#FFD700`
  - 의미: 브랜드의 가치, 유산, 에너지, 미래 지향.
  - 활용: 공식 로고 테두리, 강조 뱃지, 버튼 배경, 호버 포인트.
- **Accent Color (Action Red):** `#C40C0C`
  - 의미: 열정, 현장성, 강력한 액션. (네팔/몽골 프로젝트 로고에서 유래)
  - 활용: 프로젝트 태그, 중요 공지, 카테고리 강조 요소.
- **Neutral Surface:** `#F8F9FA`
  - 의미: 청결함, 리서치의 투명성.
  - 활용: 본문 배경색, 정보 카드 섹션.

---

## 3. Typography
현대적이고 읽기 쉬우면서도, 권위가 느껴지는 서체 조합을 사용합니다.

- **Headings (Outfit / Bricolage Grotesque):**
  - 특징: 기하학적이면서 따뜻한 느낌.
  - 활용: Title, Section Header (Bold, 700-800 weight)
- **Body (Inter / Roboto):**
  - 특징: 가독성에 최적화된 높은 완성도.
  - 활용: 본문, 설명문 (Regular/Medium, 400-500 weight)

---

## 4. Visual Language & Components

### 4.1. Glassmorphism & Soft Shadows
- **Card Design:** 완전한 흰색 배경 대신, 약간의 투명도가 들어간 Surface 색상과 부드러운 배경 블러(Backdrop Blur)를 사용하여 현대적인 깊이감을 줍니다.
- **Shadows:** 물리적으로 자연스러운 아주 부드럽고 넓게 퍼지는 그림자(Diffuse Shadows)를 사용합니다.

### 4.2. Immersive Photography
- **Grain & Texture:** 사진 위에 아주 미세한 필름 그레인 효과를 중첩하여 '현장감'과 '아날로그적 진정성'을 부여합니다.
- **Parallax Imagery:** 스크롤 시 프로젝트 사진이 미세하게 움직여 공간감을 형성합니다.

### 4.3. Micro-Animations (The "Aha" Moments)
- **Hover States:** 버튼이나 카드에 마우스를 올리면 색상이 변하는 것을 넘어, 아이콘이 부드럽게 움직이거나 이미지가 안쪽으로 살짝 줌인되는 효과.
- **Scroll Reveal:** 콘텐츠가 나타날 때 단순 페이드인이 아닌, 아래에서 위로(Staggered Reveal) 리드미컬하게 등장.

## 5. UI Layout Strategy

- **The Epic Hero:** 화면 전체를 채우는 고화질 프로젝트 영상 혹은 이미지 시퀀스. 그 위에 `Axis Gold` 색상의 굵은 타이포그래피 배치.
- **Asymmetric Grid:** 프로젝트 갤러리는 정형화된 바둑판 배열이 아닌, 잡지 레이아웃 같은 비대칭 그리드를 활용하여 예술적이고 전문적인 느낌 전달.
- **Impact Dashboard:** One Tree Planted처럼 숫자를 보여주되, 수치가 카운트업(Count-up) 되는 애니메이션과 함께 해당 데이터가 의미하는 시각적 그래픽(예: 자라나는 나무 아이콘)을 결합.

---

## 6. Admin Interface Strategy (Intuitive UX)
관리자가 디자인 가이드를 몰라도 디자인 퀄리티가 유지되도록 설계합니다.

- **WYSIWYG(What You See Is What You Get):** 웹사이트 실제 화면 위에서 바로 텍스트를 고치는 듯한 편집 경험.
- **Auto-Formatting:** 사진을 업로드하면 자동으로 위에 기술한 '필름 그레인'과 '최적화된 크롭'이 적용되는 시스템.
- **Status Toggles:** 프로젝트 모집 유무를 단순 클릭 한 번으로 변경 (Switch UI 활용).
